# Contributing to @compilr-dev Examples

Thanks for your interest in the compilr.dev examples.

## Reporting Issues

Found an issue with one of the examples? [Open an issue](https://github.com/compilr-dev/examples/issues/new/choose).

Include:
- Which example you're running
- What went wrong
- Your Node.js version and OS

## Suggesting Examples

Have an idea for a new example? Open an issue describing the use case. We're happy to add examples that demonstrate common patterns with the `@compilr-dev/agents` library.
